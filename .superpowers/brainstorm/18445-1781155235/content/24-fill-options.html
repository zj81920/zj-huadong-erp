几个实用选项，选最适合的

A

设计阶段

显示该项目的阶段列表，如「初步设计 / 施工图 / 后期服务」。直观了解项目当前在哪几个阶段。

B

计划 vs 实际进度差

如「实际 65% / 计划 50% → +15%」或「实际 30% / 计划 55% → -25%」，一眼看出差距。

C

项目类型 + 类别

如「设计 / 石油化工」，帮助区分不同业务类型的项目。

D

组合：阶段 + 进度差

两行信息：上行显示阶段标签，下行显示进度偏差。信息密度高但不拥挤。
