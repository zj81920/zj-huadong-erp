数据流 & 改造计划

数据流（页面 → API → 数据库 → 返回）

1. 页面 usePagination 提供 page, pageSize → fetch API
2. API parseInt(page) + parseInt(pageSize) → Prisma skip/take + count
3. 数据库 执行分页查询，返回 data + pagination 对象
4. 页面渲染 setPagination(res.pagination) → PaginationBar 显示

筛选联动机制

usePagination hook 提供 resetPage() 方法，筛选变化时调用：

// hook 内部实现
const handleFilterChange = useCallback((newFilters) => {
  setFilters(newFilters);
  setPage(1);  // 自动重置到第一页
}, []);

页面只需将新筛选条件传给 handleFilterChange，页码自动重置。
