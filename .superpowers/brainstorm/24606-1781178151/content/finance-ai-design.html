⚙️ 后端架构 — POST /api/ai/finance-query

后端收到用户问题后，分 3 步完成查询和回答：

① 解析用户问题 → 查询计划
→
② 执行Prisma 查询数据库
→
③ 分析AI 生成分析+图表配置

请求：

POST /api/ai/finance-query
{ "query": "最近3个月的支出按分类显示" }

响应（结构化 JSON）：

{
  "analysis": "近3个月总支出¥286,500，环比增长12.3%...",
  "chart": {
    "type": "bar",
    "title": "近3个月支出分类对比",
    "data": [...],
    "xField": "name", "yField": "value"
  },
  "table": {
    "columns": ["月份","支出金额","主要类别"],
    "rows": [...]
  },
  "cards": [
    { "label": "4月支出", "value": "¥89,200", "color": "red" }
  ]
}

📌 前端根据返回的 JSON 结构，自动渲染文字、图表、表格、卡片。 没有图表的查询（如"上季度净利润多少"）只返回 analysis 字段。
